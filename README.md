## ✨ 项目简介

**CodeNote** 是一份从 2025 年 10 月开始持续输出的个人编程知识笔记集，覆盖 **Java · Python · C++ · AI/LLM · 操作系统 · 数据库 · 计算机网络 · 编译原理** 八大技术领域，每月稳定更新。

配套的 **导航官网**（[在线预览 →](https://worldguigui.github.io/github-claw/)）提供：

- 🔍 **全文搜索**：实时过滤所有笔记标题
- 📅 **月份时间线**：按月索引，直接跳转 GitHub 原文
- 📊 **增长趋势图**：可视化笔记积累进度
- 📝 **更新日志**：追踪每一次知识沉淀

## 🌐 导航官网

官网地址：**[https://worldguigui.github.io/github-claw/](https://worldguigui.github.io/github-claw/)**

官网特性一览：

| 功能 | 说明 |
|------|------|
| 🔍 实时搜索 | 在搜索框输入关键词，即时过滤所有笔记标题 |
| 📅 月份时间线 | 按月份折叠展示，点击笔记标签直达 GitHub 原文 |
| 📊 增长趋势图 | 基于 Chart.js 的交互式折线图，可视化笔记增长 |
| 📝 更新日志 | 追踪每一次笔记新增与修改 |
| ✨ 粒子背景 | Canvas 动态粒子特效，沉浸式阅读体验 |
| 📱 响应式布局 | 移动端与桌面端自适应，支持减弱动效偏好 |
| ♿ 无障碍支持 | 语义化 HTML + ARIA 属性，阅读进度条 |

## 🚀 快速开始

本项目无需任何依赖，直接打开 `index.html` 即可本地预览官网：

```bash
# 克隆仓库
git clone https://github.com/worldguigui/github-claw.git
cd github-claw

# 用浏览器打开
open index.html   # macOS
start index.html  # Windows
```

或直接访问在线版：[https://worldguigui.github.io/github-claw/](https://worldguigui.github.io/github-claw/)
