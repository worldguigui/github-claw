## ✨ 项目简介

**CodeNote** 是一个完全开源、免费的编程知识笔记集合，由 [@worldguigui](https://github.com/worldguigui) 从 2025 年 10 月起持续维护。笔记覆盖大学核心课程与工程实践，按月整理，每月保持更新。

配套提供了一个**高颜值导航官网**（本仓库 `index.html`），采用深色主题 + 粒子背景 + 流畅动效，可在 GitHub Pages 上一键访问所有笔记。

## 🌐 导航官网功能

**官网地址：** https://worldguigui.github.io/github-claw/

| 功能 | 说明 |
|------|------|
| 🔍 **实时搜索** | 输入关键词即时过滤全部笔记标题 |
| 📊 **增长趋势** | Chart.js 可视化月度笔记增长数据 |

| 📖 **阅读进度** | 顶部进度条实时显示滚动位置 |
| 📱 **响应式** | 完美适配桌面与移动端 |
| ♿ **无障碍** | ARIA 标签完整，减少动效模式支持 |

## 🚀 快速开始

### 访问笔记

直接访问官网导航：**https://worldguigui.github.io/github-claw/**

或前往 [codenote 仓库](https://github.com/worldguigui/codenote) 按月份浏览 Markdown 文件。

### 本地运行官网

```bash
# 克隆本仓库
git clone https://github.com/worldguigui/github-claw.git

# 直接用浏览器打开
open index.html
# 或用任意静态服务器启动（推荐 VSCode Live Server）
```
