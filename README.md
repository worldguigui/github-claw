# github-claw — 个人 AI 工作空间

这是一个以 GitHub Copilot 网页版为核心的长期个人 AI 工作空间。

| 文件 / 目录 | 用途 |
|---|---|
| [`AGENTS.md`](./AGENTS.md) | 助手身份、工作规范、记忆管理、收尾动作（**从这里开始**） |
| [`MEMORY.md`](./MEMORY.md) | 跨会话长期记忆：偏好、决策、技术约定 |
| [`memory/`](./memory/) | 每日会话日志与临时记录 |

---

每次开启新的 Copilot 对话，先读 `AGENTS.md`，即可恢复上下文、延续工作。
